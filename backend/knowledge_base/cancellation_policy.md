# Cancellation Policy

This document supports appointment cancellation and rescheduling guidance only.

Patients can cancel an appointment while it is still booked. After cancellation, the appointment status becomes cancelled and the slot can become available for another patient.

Patients who need a different time should cancel the existing appointment and book a new available slot. The system should avoid double booking and should show a clear message if the selected slot is no longer available.

Emergency symptoms should not be handled by cancellation or rescheduling workflow. Patients with emergency red flags should seek emergency care immediately.
